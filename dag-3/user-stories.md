## User Stories

1. **Als gebruiker wil ik een foto kunnen uploaden zodat anderen deze kunnen zien.**
   - Prioriteit: Must
   - Acceptatiecriteria:
     - Uploadformulier accepteert alleen afbeeldingen (jpg/png/gif)
     - Foto wordt opgeslagen in de database en uploads map
     - Upload toont een succesmelding

2. **Als gebruiker wil ik een overzicht van alle foto’s zien zodat ik snel kan bladeren.**
   - Prioriteit: Must
   - Acceptatiecriteria:
     - Alle foto’s worden getoond in een grid of lijst
     - Nieuwste foto’s verschijnen bovenaan
     - Klikken op foto leidt naar detailpagina

3. **Als gebruiker wil ik een foto in detail bekijken zodat ik deze beter kan zien.**
   - Prioriteit: Must
   - Acceptatiecriteria:
     - Detailpagina toont foto en titel
     - Lijst van reacties onder de foto
     - Knoppen voor like/dislike zichtbaar

4. **Als gebruiker wil ik een foto kunnen liken zodat ik mijn waardering kan tonen.**
   - Prioriteit: Must
   - Acceptatiecriteria:
     - Like wordt in de database opgeslagen
     - Aantal likes wordt correct weergegeven
     - Eén like per gebruiker/IP

5. **Als gebruiker wil ik een reactie kunnen plaatsen onder een foto zodat ik feedback kan geven.**
   - Prioriteit: Must
   - Acceptatiecriteria:
     - Reactieformulier aanwezig
     - Reacties verschijnen direct onder de foto
     - Naam + tekst verplicht

6. **Als gebruiker wil ik het aantal likes en reacties kunnen zien zodat ik kan zien welke foto’s populair zijn.**
   - Prioriteit: Nice
   - Acceptatiecriteria:
     - Teller voor likes zichtbaar
     - Teller voor aantal reacties zichtbaar
